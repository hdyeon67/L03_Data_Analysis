# 📊 데이터 분석 1주차 복습 챌린지 (Review Challenge)

L03_Data_Analysis 강의 내용을 바탕으로 구성된 요일별 복습 연습문제 세트입니다.
 
AI로 제작된 문제로, 문제 및 정답이 정확하지 않을 수 있습니다.

## 📂 구성 안내

| 회차 | 주제 | 핵심 내용 | 챕터 난이도 |
| :--- | :--- | :--- | :--- |
| **Day 1** | Pandas 기초 | Series/DataFrame 생성, 데이터 확인 (`head`, `info`) | 하 |
| **Day 2** | 데이터 선택/필터링 | `loc`, `iloc`, Boolean Indexing (조건 필터링) | 하 |
| **Day 3** | 결측치 및 시각화 | `isnull`, `dropna`, `fillna`, Pandas 기본 그래프 | 중 |
| **Day 4** | 이상치 및 Groupby | Boxplot, IQR, `groupby` 집계 연산 | 중 |
| **Day 5** | 피벗 테이블/Matplotlib | `pivot_table`, Matplotlib 기본 구조 | 중 |
| **Day 6** | Seaborn 실전 시각화 | `scatterplot`, `barplot`, `boxplot`, `hue` 옵션 활용 | 중 |
| **Day 7** | 상관분석 및 시계열 | `corr()`, `sns.heatmap`, `resample()`, `rolling()` | 중 |
| **Final** | 종합 심화 분석 | TMDB 영화 데이터(JSON)를 활용한 전 과정 복습 | 상 |

> **참고** : 
Day별 문제 난이도 하, 중, 상으로 구분되었습니다.

특히 Final은 TMDB 영화 데이터(JSON)를 활용한 종합 심화 분석 문제입니다. 시험 문제 대비로서의 활용은 권장하지 않습니다.


## 🚀 학습 방법
1. 각 요일에 해당하는 `.ipynb` 파일을 엽니다.

3. 문제 설명을 읽고 빈 코드 셀에 정답을 작성합니다.
4. 스스로 해결하기 어려운 경우 `Answers.ipynb` (또는 제공된 정답 가이드)를 참고하세요.
